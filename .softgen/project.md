# kAIkus - AI Model Integration Platform

## Vision
Platforma spojující nejpoužívanější AI modely (OpenAI, Anthropic, Google, Mistral) prostřednictvím jednotného rozhraní. Uživatelé mohou spravovat API klíče, vytvářet konverzace a komunikovat s různými modely z jednoho místa.

## Design
- **Colors:**
  - `--primary: 262 83% 58% (electric purple)` - pro hlavní akce a headery
  - `--secondary: 280 89% 34% (deep violet)` - pro karty a sekundární prvky  
  - `--accent: 142 76% 36% (emerald)` - pro úspěšné stavy a ikony
  - `--muted: 240 5% 96% (warm gray)` - pro pozadí karet
  - `--background: 0 0% 100% (pure white)` - čisté pozadí
  - `--foreground: 240 10% 4% (near black)` - hlavní text
- **Fonts:** 
  - Heading: Plus Jakarta Sans (600, 700)
  - Body: Work Sans (400, 500, 600)
- **Style:** Moderní zen s fialovým akcentem, klidné rozhraní s jasnou hierarchií

## Features
- Dashboard s přehledem AI modelů
- Správa API klíčů pro různé poskytovatele
- Chat rozhraní pro komunikaci s modely
- Historie konverzací
- Porovnání odpovědí z různých modelů
